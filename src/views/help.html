Using this Map

This map displays the bicycle routes and features in and about the MAG region.
To view information for a particular route or item in the map pan and zoom to a location or feature of interest and click on it. Features are displayed over the background or basemap.
Popup results may include more than one page. Look for a page count on the left in the title bar and paging arrows on the right as shown below.

Map Navigation

Pan:
  • Click and drag on the map
  • Use keyboard arrow keys
Zoom In:
  • Click the Zoom In tool
  • SHIFT + drag a box in the map
  • Mouse scroll forward
Zoom Out:
  • Click the Zoom Out tool
  • SHIFT + CTRL + drag a box in the map
  • Mouse scroll reverse

Home

Zoom to the full extent of the MAG Region

Basemap Control

Click the Basemap icon to change the map background. Choices are between street maps and aerial images.

Layer List

Open the Layer List to interact with and control the display of map content. Each entry in the list is considered to be a layer. The visibility of each layer is controlled by checking (visible) or unchecking (hidden) the box to the left of the layer name.

My Location

Use the My Location button to zoom the map to an estimate of your current location. This generally works better on mobile devices, especially with GPS enabled, than on desktops.

About

The About button opens this page with map information and instructions for use.
