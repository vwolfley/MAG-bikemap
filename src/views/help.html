Using this Map

Map Navigation

Pan:
  • Click or touch the map and drag
  • Use keyboard arrow keys
Zoom:
  • Click the zoom buttons Zoom In Zoom Out
  • SHIFT + drag a box in the map
  • Scroll using the mouse
Home
  • Return to default map view

Basemap Control

Click the Basemap icon to change the map background. Choices are between street maps and aerial images.

Layer List

Open the Layer List to interact with and control the display of map content. Each entry in the list is considered to be a layer. The visibility of each layer is controlled by checking (visible) or unchecking (hidden) the box to the left of the layer name.

My Location

Use the My Location button to zoom the map to an estimate of your current location. This generally works better on mobile devices, especially with GPS enabled, than on desktops.

About

The About button opens this page with map information and instructions for use.
