Using this Map
Map Navigation
Pan:
  • Click or touch the map and drag
  • Use keyboard arrow keys
Zoom:
  • Click the zoom buttons: Zoom In Zoom Out
  • SHIFT + drag a box in the map
  • Scroll using the mouse
Home
  • Return to the default map view
Basemap Control
Toggle Basemap
Click the Basemap Toggle Button to toggle the map background between a gray basemap and satellite imagery.
Find My Location
Use the My Location button to zoom the map to your current location. This works best on mobile devices with GPS enabled.
Layer List
Open the Layers List to interact with and control the display of map content. The visibility of each layer is controlled by checking (visible) or unchecking (hidden) the box to the left of the layer name.
